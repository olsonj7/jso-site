# JSO Solutions — Site scaffold

This repository holds a minimal static website for JSO Solutions. It includes basic pages and a placeholder brand so you can preview locally.

Files added:
- `index.html` — home page
- `services.html` — services overview
- `contact.html` — contact page with simple form
- `products/index.html` — products list
- `products/linkside/` — Linkside product page and legal stubs
- `assets/css/style.css` — site stylesheet (update the color variables)
 - `assets/images/JSOSolutions1.jpg` — site logo (replace with your logo)

Preview locally:

```bash
python3 -m http.server 8000
# then open http://localhost:8000 in your browser
```
